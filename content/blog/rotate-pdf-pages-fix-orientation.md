You open a PDF and page 3 is sideways. Page 7 is upside down. The rest are fine. This happens constantly with scanned documents, merged files from different sources, and PDFs exported from phones.

The fix takes less than 30 seconds — and you don't need any software.

## Why PDF Pages Get Rotated

Understanding the cause prevents the problem next time:

- **Scanning** — Flatbed scanners and phone scanners don't always detect page orientation correctly. Landscape documents scanned in portrait mode end up sideways.
- **Merging files from different sources** — Combining a portrait contract with a landscape spreadsheet creates mixed orientations.
- **Phone photos** — When you photograph a document and convert to PDF, the phone's orientation sensor sometimes gets it wrong.
- **PDF export bugs** — Some software (especially older print drivers) exports pages with incorrect rotation metadata.
- **Different paper sizes** — Mixing A4 and Letter-size pages from different regions.

## How to Rotate PDF Pages (3 Steps)

1. Open the [Rotate PDF tool](/rotate-pdf)
2. Upload your PDF — you'll see page thumbnails
3. Click the rotation button on any page to rotate it 90° clockwise, counter-clockwise, or 180°. Then download.

You can rotate individual pages without affecting the others. If all pages need the same rotation, select all and rotate at once.

### Rotation Options

| Rotation | When to use |
|----------|-------------|
| **90° clockwise** | Page is laying on its left side |
| **90° counter-clockwise** | Page is laying on its right side |
| **180°** | Page is completely upside down |
| **No change** | Page is already correct — leave it alone |

## Working With Mixed-Orientation Documents

Some documents intentionally have mixed orientations — a portrait report with landscape tables and charts inserted between sections. This is common in:

- Annual reports (portrait text + landscape financial tables)
- Engineering documents (portrait specs + landscape diagrams)
- Research papers (portrait text + landscape data tables)

In these cases, every page should be rotated so the **content reads correctly when viewed on screen**. The mix of portrait and landscape pages is intentional and correct.

## Batch Rotation

If your entire PDF is rotated the same wrong way (e.g., everything is 90° clockwise):

1. Upload to [Rotate PDF](/rotate-pdf)
2. Select **all pages**
3. Rotate them all at once in the same direction
4. Download

This is common with scanner outputs where the entire batch was fed in the wrong orientation.

## Preventing Rotation Issues

**When scanning:**
- Feed pages in the correct orientation for your scanner
- Use your scanner's auto-orientation feature if available
- For mixed portrait/landscape pages, scan each orientation separately and merge afterward

**When photographing documents:**
- Hold your phone in portrait mode for portrait documents
- Hold your phone in landscape mode for landscape documents
- Lock your phone's orientation before photographing

**When merging:**
- Check each file's orientation before merging
- Fix any rotation issues in individual files first
- Verify the merged output page by page

## Does Rotation Affect Quality?

No. Rotation changes the page's display metadata — it tells the PDF viewer to show the page at a different angle. The actual content (text, images, vectors) is untouched. You can rotate back and forth with zero quality loss.

## FAQ

**Can I rotate just one page in a multi-page PDF?**
Yes. The tool shows thumbnails of every page — click the rotation button on only the pages that need fixing.

**Does rotating change the file size?**
No. Rotation updates a metadata flag in the PDF — the file size stays virtually identical.

**Can I rotate a password-protected PDF?**
First unlock it using [Unlock PDF](/unlock-pdf), then rotate. Re-protect afterward using [Protect PDF](/protect-pdf) if needed.

**Why does my PDF look correct on my computer but rotated when I email it?**
Different PDF viewers sometimes interpret rotation metadata differently. After rotating with the tool, the fix is universal — it works correctly in every viewer.

**Can I rotate a PDF to a custom angle (like 45°)?**
No — PDF rotation supports 0°, 90°, 180°, and 270° only. Custom angles aren't part of the PDF specification.
